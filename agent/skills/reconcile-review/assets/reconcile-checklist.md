# Reconcile Checklist

## Before Classification

- [ ] package truth is readable
- [ ] selected slice is readable
- [ ] normalized evidence is readable
- [ ] evidence maps to the selected slice
- [ ] done conditions are explicit
- [ ] verification checks are explicit

## Allowed Classification

- [ ] `accepted`
- [ ] `patch-required`
- [ ] `relaunch`
- [ ] `hold`

## Final Checks

- [ ] classification is grounded in evidence
- [ ] uncertainty is explicit
- [ ] next action is bounded and concrete
- [ ] no silent source edit is implied
- [ ] no invented owner decision is hidden
