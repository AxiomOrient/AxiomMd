# Standards

AxiomMd가 소유하는 generic 표준 정의 문서 모음.

generic contract는 이 레포가 소유하고, feature package의 제품별 스키마/shape은 product output profile이 보강한다.

## Files

- `spec-package-standard.md` — bounded feature package의 generic 기준 (required files, linkage rule, readiness rule)

## Owned Here

- generic package structure
- generic required files
- generic linkage rules
- generic readiness expectations
- generic slice concept

## Not Owned Here

- product-specific package metadata
- product-specific local readiness meaning
- product-specific execution contract
- product-specific evidence or reconcile schema

각 product repo의 output profile이 이 표준 위에 자기 profile을 덧붙인다.
